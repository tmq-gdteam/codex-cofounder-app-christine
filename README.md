# cofounder.new App Prototype

Static deployment artifact for the `cofounder.new` app UI prototype.

This folder intentionally contains only deploy-ready files:

- `index.html`
- `styles.css`
- `app.js`
- GitHub Pages workflow
- `.nojekyll`

The app is a static frontend prototype. It uses local prototype state only and does not include real AI orchestration, backend APIs, auth, billing, integrations, outreach, deployment, or external execution.
